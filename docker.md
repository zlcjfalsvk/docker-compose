# 도커 공부

### 도커 네트워크 기능

**도커의 네트워크 드라이버**

1. 브릿지(bridge) - 사용자가 정의한 브리지를 생성해 각 컨테이너에 연결
2. 호스트(host) - 호스트의 네트워크 환경
3. 논(none) - 설정 X 
4. 컨테이너(container) - 다른 컨테이너의 네트워크 환경 공유
5. 오버레이(overlay)
